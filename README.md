# 🪙 Flipen (HeadsUp)

**Flipen** is a decentralized, provably fair coin-flip game built on the **Celo** network. It allows users to double their crypto (CELO or cUSD) with a high-fidelity, real-time gaming experience optimized for **MiniPay** and mobile users.

## 🌟 Key Features

### 🛡️ Secure Native Randomness
Flipen utilizes a **Two-Step Future Block Entropy** model. Unlike simple RNG systems, Flipen splits the bet into two distinct transactions:
1.  **Commitment**: The user places a bet and "flips" the coin. The contract records the block number.
2.  **Resolution**: The user "catches" the coin in a future block. The result is generated using the hash of the commitment block, ensuring neither the player nor the contract owner can predict or manipulate the outcome.

### 💰 Multi-Asset Support
Play with the assets you already have in your wallet:
*   **Native CELO**: Direct blockchain interaction with minimal fees.
*   **cUSD (Celo Dollar)**: Stablecoin support for players who want to avoid volatility.

### 📱 Optimized for MiniPay
Designed with a mobile-first philosophy, the UI is lightweight, responsive, and features a deep integration with the MiniPay ecosystem for a "web2-like" feel on a web3 backend.

### 🛠️ Advanced Admin Dashboard
Full control for protocol owners:
*   **Bankroll Management**: Fund the contract or withdraw profits in real-time.
*   **Dynamic Limits**: Update minimum and maximum bet amounts on-the-fly.
*   **Emergency Pause**: Kill-switch to stop all games instantly if needed.

## 🏗️ Technical Stack

### **Smart Contracts (`/smc`)**
*   **Language**: Solidity ^0.8.28
*   **Framework**: Hardhat
*   **Architecture**: OpenZeppelin Transparent Upgradeable Proxy

### **Frontend (`/FE`)**
*   **Framework**: Next.js 16 (App Router)
*   **Styling**: Tailwind CSS & Framer Motion
*   **Web3**: Wagmi v3, Viem v2, RainbowKit 2.2
*   **UI Components**: Radix UI (via Shadcn)

## 🚀 Getting Started

### **1. Smart Contract Setup**
```bash
cd smc
npm install
cp .env.example .env # Add your PRIVATE_KEY and CELOSCAN_API_KEY
```

**Commands:**
*   `npm run compile`: Compile the contracts.
*   `npm run test`: Run the local test suite.
*   `npm run deploy:sepolia`: Deploy to Celo Sepolia.
*   `npm run deploy:celo`: Deploy to Celo Mainnet.

### **2. Frontend Setup**
```bash
cd FE
npm install --legacy-peer-deps
npm run dev
```

The frontend will automatically use the latest contract addresses generated by the deployment script.

## ⚖️ Game Mechanics
*   **Payout**: 1.95x (97.5% Return to Player)
*   **House Edge**: 2.5% (Used for platform maintenance and bankroll depth)
*   **Winning Logic**: `keccak256(blockhash(commitBlock), prevrandao, timestamp, playerAddress, gameId) % 2`
